# Nx Angular Barrel Error Demo

Demo Angular com Nx para reproduzir erros de dependencia ciclica via barrels (ReferenceError e TypeError), usando libs reais e um layout de app inspirado no seu todo list.

## Requisitos

- Node.js

## Comandos

- Instalar deps: `npm install`
- Rodar app: `npm run serve`
- Build dev: `npm run build`
- Graph: `npm run graph`
